# A simple showcase/scene using 3D WebGL primitives (cube, sphere, cylinders)

## Project info:
<ol>
    <li><b>Developed using Google Chrome.</b></li>
    <li>3D cube, cylinder, and sphere implemented.</li>
    <li>5 layers of hierarchical transformation</li>
    <li>5 possible interactions via transformations done for scene, alien, cylinder, alien head, and alien eye.</li>
    <li>Camera controls for pitch, yaw, and roll implemented.</li>
    <li>Perspective projection added to scene (line 65).</li>
    <li>Instructions and keystrokes for transformation and camera controls in HTML.</li>
</ol>

<b>External sources used:</b>
<ol>
    <li>Cylinder: https://www.songho.ca/opengl/gl_cylinder.html</li>
